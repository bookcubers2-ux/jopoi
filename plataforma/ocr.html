Convertir un documento escaneado en texto accesible

Muchas instituciones publican leyes como imágenes escaneadas que el lector de pantalla no puede leer. Aquí eso se corrige:

  1. Suba el archivo PDF o la foto del documento.
  2. La plataforma extrae el texto dentro de su propio dispositivo. El documento no se envía a ningún servidor.
  3. El texto queda limpio y listo para escucharlo con voz, leerlo en braille o descargarlo para impresora braille.

La primera vez se necesita internet para descargar la herramienta de lectura. Después queda guardada y funciona sin conexión.

Formatos aceptados: PDF, JPG, PNG. Si es una foto, procure que tenga buena luz.
